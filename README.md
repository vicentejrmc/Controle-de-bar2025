# Controle de Bar 2025

### Desenvolvido como uma aplicação Web na plataforma .Net Core MVC, utilizando da linguagem de C#. Desenvolvido durante o curso Full-Stack da Academia do Programador 2025



com o objetivo de auxiliar de forma simples o
gerenciamento do atendimento ao cliente. Com uma Inteface amigável e intuitiva, o sistema é ideal
para estabelecimentos de pequeno porte.
Por meio dele é possível realizar o Controle de vendas gerenciando os pedidos, 
Cadastro de produtos, Garsons e Mesas.

---

# Tecnologias Utilizadas

- .Net Core MVC
- C#
- HTML5
- CSS3
- JavaScript
- Bootstrap

---

### Funcionalidades

- Cadastro de Produtos
- Cadastro de Garçons
- Cadastro de Mesas
- Controle de Pedidos
- Controle de Vendas
- Relatórios de faturamento diario.

# Instalação

1. Clone o repositório ou baixe o código fonte.
2. Abra o terminal ou o prompt de comando e navegue até a pasta raiz
3. Utilize o comando abaixo para restaurar as dependências do projeto.
```bash
   dotnet restore
   ```

4. Em seguida, execute o comando para compilar e iniciar a aplicação.
 ```bash
   dotnet build --configuration Release
   ```
5. Para executar o projeto compilando em tempo real
```
   dotnet run --project ControleDeBar2025.WebApp
   ```

6. Para executar o arquivo compilado, navegue até a pasta 
 ./ControleDeBarWebApp/bin/Debug/net8.0/ e execute o arquivo:

 ```
   ControleDeBarWebApp.exe
   ```

7. Para modificações ajustes ou personalizações no projeto, 
 é necessário ter o Visual Studio instalado.

 # Requisitos

 	- .NET Core SDK 8.0 ou superior
    - Visual Studio 2022 ou superior (opcional, para desenvolvimento)
